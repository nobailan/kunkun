## 一、版本概述

v0.4.0 开始补全工具系统。新增文件内容搜索（grep）和精确文本替换（edit），解决 v0.3 只有全文件读写的问题。

设计原则：工具适配 DSv4 的 ThinkBlock 特性——先搜后看再改，而不是直接操作。

---

## 二、新增工具

### Grep — 文件内容正则搜索

```
grep(pattern="class ", path="src/", glob="*.py", output_mode="content")
  → 摘要表 (文件 | 匹配数) → 详细匹配 (上下文 ±2 行)
```

| 特性 | 说明 |
|------|------|
| 正则搜索 | 支持 `TODO\|FIXME`、`def function_name` 等 |
| 三种输出 | content (详细) / files_with_matches (文件列表) / count (计数) |
| 上下文行 | 可配置 ±N 行上下文 |
| DSv4 适配 | 先给结构摘要 → ThinkBlock 分析 → 再有针对性地 read_file |

### Edit — 精确字符串替换

```
edit(file_path="src/a.py", old_string="x = 1", new_string="x = 42")
  → 唯一性检查 → 替换 → 备份 → diff 摘要
```

| 特性 | 说明 |
|------|------|
| 精确匹配 | old_string 必须精确匹配（含缩进） |
| 安全保护 | 多处匹配时拒绝（除非 replace_all=True） |
| 自动备份 | 写入失败自动恢复 |
| 创建文件 | old_string 为空时创建新文件 |
| DSv4 适配 | 配合 grep 先搜后改 |

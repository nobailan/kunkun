# Kunkun v0.4.3 更新公告

> 发布日期: 2026-07-07
> 版本: v0.4.3
> 状态: 代码智能 — 符号搜索 + 定义追踪 + 引用查找

---

## 一、版本概述

v0.4.3 新增代码智能工具。Kunkun 现在可以理解代码结构——搜索符号定义、跳转到定义位置、查找引用关系。

三件套基于 Python AST 实现，零外部依赖，不需要安装 Language Server。

---

## 二、新增工具

### findsymbol — 符号搜索

```
findsymbol(query="AgentLoop", path="src/kunkun/core")
  → 类/函数/异步函数 定义列表 (文件、行号、docstring)
```

### gotodef — 跳到定义

```
gotodef(symbol="ContextManager", file_path="src/kunkun/core/context.py")
  → 符号的 class/def 声明位置 + docstring + 方法列表
```

### findrefs — 查找引用

```
findrefs(symbol="asyncio", path="src/kunkun/core")
  → 所有引用该符号的位置 (文件:行号)
```

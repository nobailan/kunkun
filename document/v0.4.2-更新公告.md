# Kunkun v0.4.2 更新公告

> 发布日期: 2026-07-07
> 版本: v0.4.2
> 状态: Agent 编排 — 子 Agent + 任务管理

---

## 一、版本概述

v0.4.2 新增 Agent 编排能力。Kunkun 现在可以拆解复杂任务、启动子 Agent 并行执行、跟踪进度。

DSv4 适配核心：**ThinkBlock 先规划 → Agent 拆解子任务 → TodoWrite 跟踪进度**。不是"收到任务直接调工具"，而是让模型的 ThinkBlock 变成任务管理器。

---

## 二、新增工具

### Agent — 子代理

```
agent(description="搜索配置文件", prompt="在项目中搜索所有 .env 和 .yaml 文件...")
  → 创建独立子 Agent → 执行 → 返回结果
```

| 特性 | 说明 |
|------|------|
| 独立上下文 | 子 Agent 拥有独立的对话窗口 |
| 类型选择 | general (通用) / explore (只读搜索) / plan (架构) |
| 并行能力 | 多个独立子任务可并行启动 |

### TodoWrite — 任务管理

```
todowrite(todos=[
  {"content": "搜索配置文件", "status": "completed", "activeForm": "搜索配置"},
  {"content": "分析依赖", "status": "in_progress", "activeForm": "分析依赖"},
  {"content": "生成报告", "status": "pending", "activeForm": "生成报告"}
])
  → 渲染任务清单 + 进度统计
```

# Kun v0.4.1 更新公告

> 发布日期: 2026-07-07
> 版本: v0.4.1
> 状态: Web 能力 — WebSearch + WebFetch

---

## 一、版本概述

v0.4.1 新增网络搜索和网页抓取能力。Kun 现在可以搜索最新文档、抓取在线内容。

---

## 二、新增工具

### WebSearch — 网络搜索

```
websearch(query="Python asyncio 并发限制", num_results=5)
  → 标题 + URL + 摘要
```

- 使用 DuckDuckGo 搜索（无需 API Key）
- 中文搜索优先
- 返回结构化结果，可直接用 webfetch 深入阅读

### WebFetch — 网页抓取

```
webfetch(url="https://docs.python.org/3/library/asyncio.html")
  → HTML → 纯文本 → 返回正文
```

- 自动检测内容类型
- HTML 转纯文本（去除标签、脚本、样式）
- 支持 prompt 参数提取相关段落
- 50K 字符截断保护
